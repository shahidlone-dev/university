# University

Modern campus ecosystem platform.

## Structure

- `campus-app/backend` → API, auth, and backend services
- `campus-app/qaaf` → mobile application built with Expo and React Native
- `campus-app/theme` → reusable design system and UI tokens

## Stack

- React Native
- Expo Router
- Node.js
- Supabase
- TypeScript

## Vision

Building a scalable campus platform combining communication, identity, and student systems.
